Default Interface

This is the first view of the form and the whole webpage when the page is first loaded. At this stage only the area of a rectangle can be calculated and the default unit is "inch" if not selected any. Since no object is selected from the "More Option", it will be considered as an undefined object.

More Option

More Icon helps to show the available objects and their operations. Clicking on an object, the available operations are visible. Choosing one of them makes the "Status Card" slide out left which holds the slected object-operation on it.

Status Card

Status card holds the selected object-operation. Keeping the cursor on it shows the formula used to calculate the current operation. On a successful calculation it blinks with green color and red color on failure for 5 seconds. Clicking on "Calculate New" button makes it disappear.

Result Area

This area is visible next to the "Geom Calculator" form after a calculation is performed. If a result is valid, i.e. all the fields in the form are positive, it appears with green color holding the result of the current object-operation and with red if any of the fields is negative. The result is rounded to 5 decimal places.

Calculate New

It appears after a calculation. Clicking on it, "Default Interface" is preserved partially by setting the unit to "inch", removing the last input values and hiding the "Result Area", "Status Card" and itself as well. Some stylings from the previous calculation - green/red sidelines on the top of the form, input field color and the form shadow color - and the "Log Button" will still be there.

Log Button

Log button slides out to the right of the form once a calculation is done and stays there unless the page is not reloaded. Clicking "Calculate New" button will not hide it. Clicking on it the "Log Table" gets expanded and the text toggles to "Hide Log" which closes the table.

Log Table

The table contains the results in descending time order. An "Invalid" result is displayed in red and white otherwise.The results are rounded to 2 decimal places and units are in short forms. A default "Calculate Area" operation has it's operation name "Undefined Object's Area" in the "Operation" column of the table.

Warning Animation

In the "More Option" area there may have object(s) whose operations are not defined and incomplete. Clicking on that object triggers a "WARNING" box to appear left to the form which goes in and out of the screen. Keeping the cursor over the box shows the corresponding warning message. It is suggested not to do any operation on that object for which a "WARNING" appears.
